# Data Structure Performance

|			  | Access		| Search	  | Insert		| Delete   |
| ----------- | ----------- | ----------- | ----------- | -------- |
| Stack		  | O(n)		| O(n)		  | O(1)		| O(1)	   |
| Queue		  | O(n)		| O(n)		  | O(1)		| O(1)     |
| Linked List | O(n)		| O(n)		  | O(1)		| O(1)     |
| Hash Table  | O(1)		| O(1)		  | O(1)		| O(1)     |
| Binary Tree | O(log n)	| O(log n)	  | O(log n)	| O(log n) |


# Stack

**Push**

O(1)

**Pop**
